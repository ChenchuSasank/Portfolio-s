Technical Skills

HTML

HTML, or Hypertext Markup Language, is the standard markup language for creating web pages. It provides the structure and layout of content on the World Wide Web.

CSS

CSS, or Cascading Style Sheets, is a style sheet language used for describing the presentation of a document written in HTML. It controls the visual aspects of web pages, such as colors, fonts, and layout.

JavaScript

JavaScript is a programming language that enables interactive web pages and dynamic content. It is commonly used for client-side scripting, allowing web developers to create engaging user experiences.

Bootstrap

Bootstrap is a front-end framework used to design and customize responsive and mobile-first websites. It provides pre-designed templates and components, making web development faster and more efficient.

Python

Python is a versatile and high-level programming language known for its simplicity and readability. It is widely used for web development, data analysis, artificial intelligence, and various other applications.

MySQL

MySQL is an open-source relational database management system (RDBMS) that uses Structured Query Language (SQL). It is commonly used for managing and querying data in web applications and other software systems.
